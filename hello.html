Hello world

  1. First you need to create your own turtle. Lets start by giving a name to your turtle, write the name of your turtle in Line 2. Example (Max etc)
  2. Now you need to call the turtle library. Add Turtle() in line number 2. *Add this:* Max = Turtle()
  3. Lets choose a turtle shape for you turtle. To do so go to Line 3 and add "Max.shape('turtle')"
  4. Add a color to your turtle. Go to line 4 and add "Max.color('red')". You can choose any color you wish
  5. Click the run button to see your turtle
  6. It's time to make our turtle move, when the turtle move it draws a line behind it. To make the turtle move, add the forward command and give it the length to move. Go to line 5 and add *Max.forward(100)*, remember the bigger the number the longer the line will be
  7. Click RUN to draw a line
